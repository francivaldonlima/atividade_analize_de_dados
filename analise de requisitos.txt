ETAPA 2: CRIAR O PROGRAMA PYTHON 

O que o programa DEVE FAZER: 

1️
⃣ LEITURA DOS DADOS (obrigatório) 

python 


# ------------------------------------------------- 

# 1. LER OS DADOS DO ARQUIVO JSON 

# ------------------------------------------------- 

import json 

 

# Abrir e ler o arquivo JSON usando with open 

with open('dados.json', 'r', encoding='utf-8') as arquivo: 

 dados = json.load(arquivo) 

 

print(f"\n✅ Dados carregados com sucesso!") 

print(f"📊 Total de registros: {len(dados)}")

🆘 DICAS PARA O SUCESSO 


Antes de começar: 

1. Escolham um tema simples - vendas é o mais fácil 
2. Testem o JSON - coloquem no ChatGPT para validar 
3. Dividam as tarefas: 
○ Um cuida do código Python 
○ Outro cuida dos dados e documentação 





Durante o desenvolvimento: 

1. Testem passo a passo: 
○ Primeiro: só leiam e exibam os dados 
○ Segundo: calculem uma métrica 
○ Terceiro: calculem as demais 



2. Use print() para debug - vejam o que está acontecendo 
3. Salvem frequentemente - não percam o trabalho 


Para as percepções: 

1. Olhem os números calculados - as percepções devem vir deles 
2. Sejam específicos - não digam só "teve muitas vendas" 
3. Conectem ideias - ex: "o vendedor X vendeu mais porque..."
